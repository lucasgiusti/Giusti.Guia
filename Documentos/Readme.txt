Projeto: Giusti.Guia

Como instalar:
01 - criar banco de dados com o nome GiustiGuia.
02 - executar o script BaseDeDados\create.sql para criação das tabelas e dados
     no banco de dados GiustiGuia.
03 - alterar a chave ConexaoBanco do arquivo Giusti.Guia\Giusti.Guia.Web\Web.config 
     apontando o usuário e senha de acesso ao banco de dados;
